# 코딩 연습 - LINQ (메서드 구문과 쿼리 구문)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을
사용할 수 있습니다.

---

## 1. 메서드 구문 기본 사용법

정수 배열 `{ 1, 2, 3, 4, 5 }`에서 `Where`를 사용하여 3보다 큰 숫자만 필터링하고
출력하세요.

**실행 결과**

```
4
5
```

---

## 2. Where - 필터링

문자열 배열 `{ "Tom", "Dick", "Harry", "Mary", "Jay" }`에서 `"a"`가 포함된 이름만 
필터링하여 출력하세요.

**실행 결과**

```
Harry
Mary
Jay
```

---

## 3. Select - 변환

문자열 배열 `{ "Tom", "Dick", "Harry" }`의 각 이름을 `Select`로 대문자로 변환하여 
출력하세요.

**실행 결과**

```
TOM
DICK
HARRY
```

---

## 4. OrderBy - 오름차순 정렬

문자열 배열 `{ "Red", "Green", "Blue" }`를 `OrderBy`로 알파벳 순 오름차순 정렬하여 
출력하세요.

**실행 결과**

```
Blue
Green
Red
```

---

## 5. OrderByDescending - 내림차순 정렬

정수 배열 `{ 3, 1, 4, 1, 5, 9, 2, 6 }`을 `OrderByDescending`으로 내림차순 정렬하여 
공백으로 구분하여 출력하세요.

**실행 결과**

```
9 6 5 4 3 2 1 1
```

---

## 6. 쿼리 연산자 체이닝

문자열 배열 `{ "Tom", "Dick", "Harry", "Mary", "Jay" }`에서 `"a"` 포함 필터링 → 
길이순 정렬 → 대문자 변환을 체이닝하여 출력하세요.

**실행 결과**

```
JAY
MARY
HARRY
```

---

## 7. 쿼리 구문 기본 구조

정수 배열 `{ 1, 2, 3, 4, 5 }`에서 쿼리 구문(`from ... where ... select`)으로 짝수만 
선택하여 출력하세요.

**실행 결과**

```
2
4
```

---

## 8. 쿼리 구문 필터링과 정렬

문자열 배열 `{ "Tom", "Dick", "Harry", "Mary", "Jay" }`에서 쿼리 구문으로 `"a"` 포함 
필터링 → 길이순 정렬 → 대문자 변환을 수행하여 출력하세요.

**실행 결과**

```
JAY
MARY
HARRY
```

---

## 9. 메서드 구문과 쿼리 구문 비교

정수 배열 `{ 3, 2, 1, 4, 5 }`에서 홀수만 선택하고 오름차순 정렬하는 쿼리를 
메서드 구문과 쿼리 구문으로 각각 작성하세요. 두 결과를 각각 공백으로 구분하여 
출력하세요.

**실행 결과**

```
메서드 구문:
1 3 5
쿼리 구문:
1 3 5
```
