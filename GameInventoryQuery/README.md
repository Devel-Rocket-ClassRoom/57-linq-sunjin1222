# 게임 인벤토리 조회

게임 인벤토리의 아이템 데이터를 **쿼리 구문(Query Syntax)**으로 
조회하는 프로그램을 작성합니다.

## GameItem 클래스

| 속성 | 타입 | 설명 |
|------|------|------|
| Name | string | 아이템 이름 |
| Type | string | "무기" / "방어구" / "소비" |
| Grade | string | "일반" / "희귀" / "전설" |
| Price | int | 가격 |
| Quantity | int | 수량 |

## 아이템 목록

| 이름 | 타입 | 등급 | 가격 | 수량 |
|------|------|------|------|------|
| 나무 검 | 무기 | 일반 | 100 | 3 |
| 강철 대검 | 무기 | 희귀 | 800 | 1 |
| 용의 검 | 무기 | 전설 | 5000 | 1 |
| 가죽 갑옷 | 방어구 | 일반 | 200 | 2 |
| 미스릴 갑옷 | 방어구 | 희귀 | 1200 | 1 |
| 드래곤 갑옷 | 방어구 | 전설 | 8000 | 0 |
| 체력 물약 | 소비 | 일반 | 50 | 10 |
| 마나 물약 | 소비 | 일반 | 80 | 5 |
| 고급 물약 | 소비 | 희귀 | 500 | 0 |
| 엘릭서 | 소비 | 전설 | 3000 | 2 |

## 구현할 쿼리 (모두 쿼리 구문 사용)

1. 가격이 500 이상인 아이템을 가격 내림차순으로 조회
2. "무기" 타입 아이템만 등급순(일반 → 희귀 → 전설)으로 조회
3. `let` 절로 총 가치(Price x Quantity)를 계산하고, 총 가치가 1000 이상인 아이템 조회
4. 수량이 0인 품절 아이템을 이름순으로 조회
5. "전설" 등급 아이템의 이름과 가격만 추출 (익명 타입 `select new { }`)

## 예상 실행 결과

```

엘릭서 - 소비 - 3000원

고급 물약 - 소비 - 500원

나무 검 - 일반 - 100원
강철 대검 - 희귀 - 800원
용의 검 - 전설 - 5000원

=== 쿼리 3: 총 가치 1000 이상 ===
용의 검 - 총 가치: 5000

엘릭서 - 총 가치: 6000

=== 쿼리 4: 품절 아이템 (이름순) ===
고급 물약

=== 쿼리 5: 전설 등급 (이름과 가격) ===
용의 검 - 5000원

```
